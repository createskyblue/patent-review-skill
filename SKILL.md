# 专利审核

## 审核流程

### 1. 确定专利类型
- **发明专利**：技术方案类，包括方法、算法、产品结构
- **实用新型**：产品形状、构造改进，不需要算法
- **外观设计**：产品外观造型、图案

### 2. 核心审核要点

#### 逻辑合理性检查
- 背景技术描述是否准确，现有技术存在的问题是否真实
- 发明目的与解决的技术问题是否对应
- 技术方案是否能够解决所述技术问题
- 实施方式是否有逻辑矛盾

#### 技术可行性检查
- 技术手段是否在现有技术基础上可以实现
- 关键技术参数是否合理
- 原理描述是否与物理/化学原理相符

#### 创新点评估
- 创新点是否清晰、具体、可实现
- 是否存在与现有技术的实质性区别
- 保护范围是否适当

### 3. 常见问题识别

| 问题类型 | 检查点 |
|---------|--------|
| 逻辑漏洞 | 前后矛盾、因果倒置、跳跃性结论 |
| 技术错误 | 违反基本原理、参数不合理、无法实现 |
| 描述不足 | 核心方案模糊、实施例太少 |
| 保护不当 | 范围过宽或过窄、缺乏层次 |

## 使用方式

用户提供专利文档后，直接按照上述审核流程进行审核，无需额外步骤。
